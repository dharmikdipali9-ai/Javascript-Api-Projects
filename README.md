# 🐱 Random Cat Facts Generator

A fun and interactive web app that generates random cat facts and random cat images with just one click!  
Perfect for cat lovers and anyone who enjoys learning interesting facts. 🐾

## ✨ Features
🐱 Generates random cat facts using a public API  
📡 Uses JavaScript Fetch API to retrieve data  
⚡ Built with Async / Await for smooth API handling  
🌙 Dark mode support for better user experience  
🎨 Clean and simple UI

## 🛠️ Technologies Used
💻 HTML  
🎨 CSS  
⚡ JavaScript  
🌐 Fetch API  

## 🚀 What I Learned
✔ Working with APIs in JavaScript  
✔ Fetching and displaying dynamic data  
✔ Using Async / Await  
✔ DOM manipulation
